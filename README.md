# Fine-Tuned DenseNet201 + PCA + SVM Brain Tumor Classification

A hybrid deep learning + machine learning framework for multi-class brain tumor classification using MRI images.

This project combines:
- Fine-Tuned DenseNet201
- PCA-based dimensionality reduction
- SVM classification

to achieve highly accurate and balanced brain tumor prediction.

---

# Overview

Traditional CNN-based medical image classifiers often suffer from:
- overfitting
- poor generalization
- high computational complexity

This project implements a hybrid pipeline inspired by recent research in medical AI, where:
- DenseNet201 extracts deep MRI features
- PCA reduces redundant dimensions
- SVM performs robust classification
